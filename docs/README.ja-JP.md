## :gift: 利点 

1. シームレスで自動的な GPU 切り替え
2. すべてのパフォーマンスモードを電力制限とファンカーブで完全にカスタマイズ可能
3. 軽量。実行するのは単一の exe ファイルだけ。システムには何もインストールしません。
4. シンプルでクリーンなネイティブ UI で、すべての設定に簡単アクセス
5. FN ロックとカスタムホットキー

<img width="1960" alt="G-Helper Screenshot" src="https://github.com/user-attachments/assets/9376fe90-fbb6-420a-abbd-0186189665e1" />

### :zap: 機能

1. パフォーマンスモード：サイレント - バランス - ターボ（内蔵、デフォルトのファンカーブ付き）
2. GPU モード：エコ - スタンダード - アルティメット - オプティマイズド
3. ディスプレイオーバードライブ対応の画面リフレッシュレート制御
4. 各パフォーマンスモードごとのカスタムファンカーブエディタ、電力制限、ターボブースト選択
5. アニメマトリックスまたはスラッシュライティングの制御（アニメーション GIF、時計、オーディオビジュアライザーを含む）
6. RGB バックライトのアニメーションモードとカラー
7. ホットキーの処理
8. CPU / GPU 温度、ファン回転数、バッテリー状態の監視
9. バッテリーの健康を保つための充電上限
10. Nvidia GPU のオーバークロックとアンダーボルト
11. XG Mobile の制御
12. AMD CPU のアンダーボルトと温度制限
13. BIOS とドライバの更新
14. Asus マウスの制御
15. Mini-LED マルチゾーン切り替え
16. フリッカーフリー調光とビジュアルモード

### :gear: 自動化
- バッテリー駆動時 / 電源接続時のパフォーマンスモード切り替え
- オプティマイズド GPU モード - バッテリー駆動時は dGPU を無効化し、電源接続時に有効化
- 自動画面リフレッシュレート（バッテリー駆動時は 60Hz、電源接続時は最大 Hz）
- バッテリー駆動時 / 電源接続時のキーボードバックライトのタイムアウト

### :rocket: パフォーマンスモード
>[!NOTE]
>すべてのモードは、デフォルトのファンカーブと電力制限とともに **BIOS に組み込まれて** おり、Armoury Crate のものと **まったく同じ** です。

各デフォルトの BIOS モードは、対応する [Windows 電源モード](https://support.microsoft.com/en-us/windows/change-the-power-mode-for-your-windows-pc-c2aff038-22c9-f46d-5ca0-78696fdf2de8) とペアになっています。この設定は ``ファン + 電源`` で調整できます。

1. BIOS の `サイレント` + `最も電力効率が良い` 電源モード
2. BIOS の `バランス`（AC 接続時は Performance）+ `バランス` 電源モード
3. BIOS の `ターボ` + `最も高いパフォーマンス` 電源モード

### :video_game: GPU チューニング

各モードで、コアクロック / メモリクロックのオフセット、GPU 電力、Dynamic Boost、温度制限など、専用 GPU を用途に合わせて調整できます。
<img width="2354" alt="GPU Tweaking" src="https://github.com/user-attachments/assets/48ce19a6-d149-40ed-9145-23f974c513f7" />

### 💻 GPU モード

1. **エコ**：低電力の内蔵 GPU のみ有効。iGPU が内蔵ディスプレイを駆動します
2. **スタンダード**（MS Hybrid）：iGPU と dGPU が有効。iGPU が内蔵ディスプレイを駆動します
3. **アルティメット**：iGPU と dGPU が有効で、dGPU が内蔵ディスプレイを駆動します（2022 年以降のモデルで対応）
4. **オプティマイズド**：バッテリー駆動時は dGPU を無効化（エコ）、電源接続時に有効化（スタンダード）

### 🎮 メトリクスオーバーレイ
ゲーム画面の上に **FPS、CPU / GPU の温度・使用率・電力** をリアルタイムで表示する内蔵のゲーム内オーバーレイ（OSD）です。追加ツールは不要です。DX10 以降のゲームに対応します。排他的フルスクリーンを使用する古いタイトルでは表示されない場合があるため、ゲームをウィンドウ / ボーダーレスフルスクリーンモードに設定してください。

- ``Ctrl + Shift + Alt + O`` - ゲーム内オーバーレイの ON / OFF を切り替え
- ``Ctrl + Shift + Alt + マウスドラッグ`` - オーバーレイを移動
- ``Ctrl + Shift + Alt + マウスクリック`` - モード切り替え（ライト / デフォルト / フル）
- ``Ctrl + Shift + Alt + ホイール`` - オーバーレイのサイズ変更
- ``Ctrl + Shift + Alt + ホイールクリック`` - オーバーレイのサイズをリセット

> [!NOTE]
> ### 🔖 重要なお知らせ
> G-Helper はオペレーティングシステム、ファームウェア、ドライバの **いずれでもありません**。お使いのハードウェアをリアルタイムで「動作」させることは **一切ありません**。
> 
> これは、メーカーが作成し（BIOS に保存されている）あらかじめ定義された動作モードの 1 つを選択し、オプションで（！）お使いのデバイスに既に存在する設定を行うためのアプリにすぎません。Armoury Crate と同じ仕組みです。Armoury Crate が使用しているのと同じ Asus System Control Interface「ドライバ」を使って、これらを実現します。
> 
> Armoury Crate と同じモード / 設定を使用する場合、デバイスのパフォーマンスや動作に違いはありません。
> 
> ノートパソコンにおける G-Helper の役割は、テレビにおけるリモコンの役割に似ています。

### 使用ライブラリとプロジェクト
- [Linux Kernel](https://github.com/torvalds/linux/blob/master/include/linux/platform_data/x86/asus-wmi.h) - ASUS ACPI/WMI インターフェースの基本的なエンドポイント
- [NvAPIWrapper](https://github.com/falahati/NvAPIWrapper) - Nvidia API へのアクセス
- [Starlight](https://github.com/vddCore/Starlight) - アニメマトリックスの通信プロトコル
- [UXTU](https://github.com/JamesCJ60/Universal-x86-Tuning-Utility) - Ryzen SMU を使用したアンダーボルト用エンドポイント
- [PawnIO](https://github.com/namazso/PawnIO) - RyzenSMU へのアクセス
- [AsusCtl](https://gitlab.com/asus-linux/asusctl) - インスピレーションと一部のリバースエンジニアリング
