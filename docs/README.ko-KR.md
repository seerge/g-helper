### :rocket: 성능 모드
>[!NOTE]
>모든 모드는 기본 팬 곡선 및 전력 제한과 함께 **BIOS에 내장되어** 있으며, Armoury Crate의 것과 **동일합니다**.

각 기본 BIOS 모드는 해당하는 [Windows 전원 모드](https://support.microsoft.com/en-us/windows/change-the-power-mode-for-your-windows-pc-c2aff038-22c9-f46d-5ca0-78696fdf2de8)와 짝을 이룹니다. 이 설정은 ``팬 + 전원``에서 조정할 수 있습니다.

1. BIOS의 `사일런트` + `최고의 전력 효율` 전원 모드
2. BIOS의 `밸런스드` (AC 연결 시 Performance) + `균형 잡힌` 전원 모드
3. BIOS의 `터보` + `최고 성능` 전원 모드

### 💻 GPU 모드

1. **에코**: 저전력 내장 GPU만 활성화되며, iGPU가 내장 디스플레이를 구동합니다
2. **표준** (MS Hybrid): iGPU와 dGPU가 활성화되며, iGPU가 내장 디스플레이를 구동합니다
3. **얼티밋**: iGPU와 dGPU가 활성화되지만, dGPU가 내장 디스플레이를 구동합니다 (2022년 이후 모델 지원)
4. **최적화**: 배터리 사용 시 dGPU를 비활성화(에코)하고, 전원 연결 시 활성화(표준)합니다

### 🎮 메트릭 오버레이
게임 화면 위에 실시간 **FPS, CPU / GPU 온도, 사용률, 전력**을 표시하는 내장 게임 내 오버레이(OSD)입니다. 별도 도구가 필요 없습니다. DX10 이상 게임에서 작동합니다. 전체 화면 독점 모드를 사용하는 구형 게임에서는 보이지 않을 수 있으므로 게임을 창 / 테두리 없는 전체 화면 모드로 설정하세요.

- ``Ctrl + Shift + Alt + O`` - 게임 내 오버레이 켜기 / 끄기
- ``Ctrl + Shift + Alt + 마우스 드래그`` - 오버레이 이동
- ``Ctrl + Shift + Alt + 마우스 클릭`` - 모드 전환 (Light / Default / Full)
- ``Ctrl + Shift + Alt + 휠`` - 오버레이 크기 조정
- ``Ctrl + Shift + Alt + 휠 클릭`` - 오버레이 크기 초기화

> [!NOTE]
> ### 🔖 중요 안내
> G-Helper는 운영 체제, 펌웨어, 드라이버가 **아닙니다**. 어떤 식으로든 하드웨어를 실시간으로 「구동」하지 **않습니다**.
> 
> 이 앱은 제조사가 만들어 (BIOS에 저장한) 미리 정의된 작동 모드 중 하나를 선택하고, 선택적으로(!) 사용자 기기에 이미 존재하는 일부 설정을 조정할 수 있게 해주는 애플리케이션일 뿐입니다. Armoury Crate와 동일한 방식이며, Armoury Crate가 사용하는 것과 동일한 Asus System Control Interface 「드라이버」를 사용하여 이를 구현합니다.
> 
> Armoury Crate와 동일한 모드/설정을 사용하면 기기의 성능이나 동작에 차이가 없습니다.
> 
> 노트북에서 G-Helper의 역할은 TV의 리모컨과 비슷합니다.
