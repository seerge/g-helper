## :gift: Přednosti 

1. Plynulé a automatické přepínání GPU
2. Všechny provozní režimy jsou plně přizpůsobitelné, včetně nastavení výkonových limitů a křivek ventilátorů
3. Nenáročný. Stačí spustit jediný soubor EXE. Do systému se nic neinstaluje.  
4. Jednoduché a přehledné nativní UI se snadným přístupem ke všem nastavením
5. FN-Lock a vlastní klávesové zkratky

### :zap: Vlastnosti

1. Provozní režimy: Tichý – Vyvážený – Turbo (přednastavené výrobcem s výchozími křivkami otáček ventilátorů)
2. Režimy GPU: Eco – Standardní – Ultimátní – Optimalizovaný
3. Možnost rychle měnit obnovovací frekvenci obrazovky (60 Hz, 120 Hz, 144 hz, ...), včetně funkce Overdrive (OD) 
4. Editor vlastních křivek ventilátorů, možnost nastavit výkonové limity a funkci Turbo Boost u každého provozního režimu
5. Ovládání osvětlení Anime Matrix nebo Slash včetně animovaných GIFů, hodin a vizualizéru zvuku
6. Režimy a barvy animace podsvícení
7. Ovládání klávesových zkratek
8. Sledování teploty CPU a GPU, otáček ventilátorů a stavu baterie
9. Omezovač nabití baterie pro zachování jejího zdraví
10. Přetaktování a podvoltování NVidia GPU
11. Ovládání XG Mobile
12. Podvoltování AMD CPU
13. Aktualizace BIOSu a ovladačů
14. Nastavení myší Asus
15. Přepínač vícezónového mini-LED podsvícení
16. Flicker-free dimming a vizuální režimy

### :gear: Automatizace
- Přepínání režimu výkonu při provozu na baterii nebo při připojení k napájení
- Optimalizovaný režim GPU – při provozu na baterii deaktivuje dGPU a při připojení k napájení ji aktivuje
- Automatická obnovovací frekvence obrazovky (60 Hz při provozu na baterii a maximální Hz při připojení k napájení)
- Časový limit podsvícení klávesnice při provozu na baterii nebo při připojení k napájení

### :rocket: Provozní režimy

<img align="right" width="300" src="https://github.com/seerge/g-helper/assets/5920850/3e119674-db8d-486b-aa65-2bf9b61f9aa6">

Všechny režimy jsou **integrovány v BIOSu** spolu s výchozími křivkami otáček ventilátorů a výkonovými limity a jsou **stejné** jako v aplikaci Armoury Crate.

Každý režim BIOSu je spárován s odpovídajícím výkonovým režimem systému Windows. Toto nastavení můžete upravit v části ``Ventilátory + výkon``.

1. **Tichý** v BIOSu + výkonový režim **Nejlepší efektivita napájení** 
2. **Vyvážený** (Výkon při napájení ze sítě) v BIOSu + výkonový režim **Vyvážený**
3. **Turbo** v BIOSu + výkonový režim **Nejlepší výkon**
   

### :video_game: Režimy GPU

1. **Eco**: povolena pouze iGPU s nízkou spotřebou, vestavěný displej je řízen iGPU
2. **Standardní** (MS Hybrid): povolena iGPU i dGPU, vestavěný displej je řízen iGPU
3. **Ultimátní**: povolena iGPU i dGPU, ale vestavěný displej je řízen dGPU (podporováno u modelů z roku 2022 a novějších)
4. **Optimalizovaný**: deaktivuje dGPU při provozu na baterii (Eco) a aktivuje ji při připojení k napájení (Standardní)

### ⌨️ Klávesové zkratky

- ``Fn + F5 / Fn + Shift + F5`` - Přepínání provozních režimů vpřed / Zpět
- ``Ctrl + Shift + F5 / Ctrl + Shift + Alt + F5`` - Přepínání provozních režimů vpřed / Zpět
- ``Ctrl + Shift + F12`` - Otevření okna G-Helperu
- ``Ctrl + M1 / M2`` - Snížení / Zvýšení jasu displeje
- ``Shift + M1 / M2`` - Snížení / Zvýšení jasu podsvícení
- ``Fn + C`` - Fn-Lock
- ``Fn + Ctrl +  F7 / F8`` / ``Ctrl + Shift + Alt +  F7 / F8`` - Snížení / Zvýšení Flicker-free dimmingu
- ``Fn + Shift + F7 / F8`` - Snížení / Zvýšení jasu Matrix / Slash podsvícení
- ``Fn + Shift + F7 / F8`` / ``Ctrl + Shift + Alt +  F7 / F8`` - Snížení / Zvýšení jasu Screenpadu
- ``Ctrl + Shift + F20`` - Vypnutí mikrofonu
- ``Ctrl + Shift + Alt + F13`` - přepnutí obnovovací frekvence displeje
- ``Ctrl + Shift + Alt + F14`` - Eco režim GPU
- ``Ctrl + Shift + Alt + F15`` - Standardní režim GPU
- ``Ctrl + Shift + Alt + F16`` - Tichý
- ``Ctrl + Shift + Alt + F17`` - Vyvážený
- ``Ctrl + Shift + Alt + F18`` - Turbo
- ``Ctrl + Shift + Alt + F19`` - Vlastní 1 (pokud existuje)
- ``Ctrl + Shift + Alt + F20`` - Vlastní 2 (pokud existuje)

> [!NOTE]
> ### 🔖 Důležité upozornění
> G-Helper **NENÍ** operační systém, firmware ani ovladač. V žádném případě nijak **NEŘÍDÍ** váš hardware v reálném čase.
> 
> Jedná se o aplikaci, která vám umožňuje vybrat jeden z předdefinovaných provozních režimů vytvořených výrobcem (a uložených v BIOSu) a případně (!) upravit některá nastavení, která již ve vašem zařízení existují, stejně jako to umí Armoury Crate. K tomuto účelu využívá „ovladač“ z Asus System Control Interface, který k tomuto účelu používá i Armoury Crate.
> 
> Pokud použijete stejný režim/nastavení jako v aplikaci Armoury Crate, výkon ani chování vašeho zařízení se nebude lišit.
> 
> Funkce G-Helperu pro váš notebook je podobná funkci dálkového ovladače pro váš televizor.
